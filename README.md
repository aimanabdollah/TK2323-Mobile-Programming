# TK2323 Mobile Programming
This repository is about TK2323 Mobile Programming - Lab Assignment that consists of 7 deliverables based on selected topic during my study.

## Synopsis Course
This course teaches the fundamental of the mobile app development using Android platform. Topics include important elements in the android app development such as user interface design, user interface building, recycler view, data handling, fragment, API and sensor. This course focuses heavily on in-lab participation, out-of-class assignments, and mobile app development project. Students are expected to work on a project to produce a mobile app. Students should already have a familiarity with Java, an understanding of basic object-oriented programming, basic algorithms and data structures. Course projects will be coded using Android Studio in Java for the Android platform.
